# 五、YD-3414 自动下载电路接线
1. 输入信号：CH340C DTR#、RTS#
2. 输出控制：
   - Q1 NPN三极管基极：控制ESP32 EN引脚
   - Q2 NPN三极管基极：控制ESP32 IO0引脚
3. 三极管统一发射极接GND，集电极分别接EN、IO0
4. 工作逻辑：下载时三极管导通，拉低EN/IO0进入烧录模式；正常运行三极管截止

# 九、按键电路接线
1. SW1复位键：一端ESP32 EN，一端GND；搭配R4(1K)+C1(1uF)RC消抖电路
2. SW2手动BOOT键：一端ESP32 IO0，一端GND
3. SW3电池总开关：串联在CN2电池供电回路

# 十、接线核心注意事项
1. 电源区分：ESP32、CH340、传感器仅支持3.3V；灯带、风扇、蜂鸣器、Cat1使用5V
2. 串口必须交叉：芯片TX对接另一芯片RX
3. 所有模块GND单点共地，避免模拟数字干扰
4. MOS管栅极必须保留下拉电阻，防止灯带微亮
5. CC引脚5.1K下拉电阻不可省略，否则Type-C识别异常
6. CH340供电只能接3.3V，接入5V会烧毁芯片
7. 免费NC空引脚无需接线，预留硬件升级拓展
