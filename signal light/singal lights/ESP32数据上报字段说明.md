# ESP32 应急指示灯 — MQTT 数据上报字段说明

## 数据格式

MQTT 主题：`indicator_light/data`，每 10 秒上报一次 JSON 数据。

```json
{
  "D8:BC:38:78:21:98": {
    "temp": 25.0,
    "hum": 50.0,
    "AQI": 1,
    "TVOC": 0,
    "ECO2": 400,
    "electricity": 791,
    "ESP32_fires_flag": false,
    "ESP32_electricity_flag": false
  }
}
```

外层 Key 为设备 MAC 地址，用于区分不同节点。

---

## 传感器字段

| 字段 | 类型 | 说明 | 正常范围 |
|------|------|------|----------|
| `temp` | float | 环境温度（°C） | 15~35°C |
| `hum` | float | 环境湿度（%） | 30~70% |
| `AQI` | int | 空气质量指数 | 0~50（优） |
| `TVOC` | int | 总挥发性有机化合物浓度（ppb） | < 220 ppb |
| `ECO2` | int | 等效二氧化碳浓度（ppm） | 400~1000 ppm |
| `electricity` | int | 电流传感器 ADC 采样值（0~4095） | > 400（有电） |

### 字段详细说明

- **temp / hum**：由 AHT20 温湿度传感器采集，反映室内环境状态。
- **AQI**：空气质量综合指标，0 表示空气优良。
- **TVOC**：总挥发性有机化合物浓度，主要来源于装修材料、清洁剂等，浓度过高影响健康。
- **ECO2**：等效二氧化碳浓度，用于判断室内通风情况，密闭空间浓度会升高。
- **electricity**：电流传感器 ADC 采样值，通过对 `electricity_pin`（GPIO36）引脚进行 8 次采样取平均。值 > 400 表示市电正常，< 400 表示断电。

## 警报字段

### ESP32_fires_flag — 火灾报警

| 值 | 含义 |
|----|------|
| `true` | 火灾报警已触发 |
| `false` | 正常状态 |

**触发场景**：远程服务器/前端通过 MQTT 下发火灾报警指令，ESP32 订阅的主题 `indicator_light/command` 收到包含 `"ESP32_fires": 1` 的 JSON 数据时触发。

**触发后的动作**：
1. 三个方向（左/中/右）的箭头 LED 全部切换为动态闪烁模式，提示紧急疏散
2. 蜂鸣器持续鸣响，发出声音警报
3. 场效应管（PWM）输出 500，开启高亮警示灯
4. 数据上报中 `ESP32_fires_flag` 变为 `true`

**解除方式**：远程服务器下发 `"ESP32_fires": 0` 指令，或系统重启。

---

### ESP32_electricity_flag — 电流报警（断电报警）

| 值 | 含义 |
|----|------|
| `true` | 正常供电 |
| `false` | 断电报警已触发 |

**触发场景**：ESP32 本地检测到市电断开时自动触发。判断逻辑：电流传感器 ADC 采样值 < 400 时认为断电。

**触发后的动作**：
1. 三个方向（左/中/右）的箭头 LED 全部切换为动态闪烁模式，提示用户疏散
2. 蜂鸣器持续鸣响，发出声音警报
3. 场效应管（PWM）输出 250，开启备用照明灯
4. 数据上报中 `ESP32_electricity_flag` 变为 `false`

**恢复方式**：市电恢复后（电流值 > 400），系统自动解除报警，恢复正常状态。

**设计目的**：应急指示灯需要在断电时自动切换到电池供电，并通过报警提醒用户注意。这是消防应急灯的核心功能，确保火灾等紧急情况下即使断电，指示灯仍能引导人员安全疏散。

---

## 警报对比

| 对比项 | 火灾报警 | 电流报警 |
|--------|----------|----------|
| 触发方式 | 远程 MQTT 指令下发 | 本地 ADC 检测断电 |
| 判断条件 | 收到 `"ESP32_fires": 1` | 电流值 < 400 |
| PWM 输出 | 500（高亮） | 250（备用灯） |
| 恢复方式 | 收到 `"ESP32_fires": 0` | 市电恢复自动恢复 |
| 应用场景 | 火灾预警、紧急疏散 | 市电断电、应急照明 |
