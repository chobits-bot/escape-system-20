# 火灾逃生系统 - 服务器操作指南

## 连接服务器

```bash
ssh ubuntu@42.193.218.29
```

## 启动服务流程

### 1. 启动 Docker 容器

```bash
cd ~/escape-project
docker compose up -d
```

### 2. 启动算法服务

```bash
cd ~/escape-project/aiot/algorithm-service
nohup python3 main.py > /tmp/algo.log 2>&1 &
```

### 3. 启动前端（使用 tmux）

```bash
tmux new -d -s frontend "cd ~/escape-project/aiot/aiot-vue && npm run dev -- --host 0.0.0.0 --port 5173"
```

## 常用命令

```bash
# 查看所有容器状态
docker ps

# 查看后端日志
docker logs aiot-backend --tail 20

# 查看MQTT连接状态
docker exec -it $(docker ps --filter "name=emqx" --format "{{.Names}}") emqx ctl clients list

# 检查MySQL数据
docker exec -it $(docker ps --filter "name=mysql" --format "{{.Names}}") mysql -u root -paiot123456 aiot -e "SELECT floor_id, COUNT(*) FROM graph_node GROUP BY floor_id;"

# 检查算法服务
curl http://localhost:8090/health
```

## 服务器重启后恢复

```bash
# 1. 启动Docker
cd ~/escape-project
docker compose up -d

# 2. 清理端口冲突（如有）
sudo fuser -k 6379/tcp 2>/dev/null
sudo fuser -k 3306/tcp 2>/dev/null
docker compose up -d

# 3. 启动算法服务
cd ~/escape-project/aiot/algorithm-service
nohup python3 main.py > /tmp/algo.log 2>&1 &

# 4. 启动前端
tmux new -d -s frontend "cd ~/escape-project/aiot/aiot-vue && npm run dev -- --host 0.0.0.0 --port 5173"
```
