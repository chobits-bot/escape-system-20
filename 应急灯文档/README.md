## 项目说明

本文件夹是 AIoT 智能消防应急灯项目的完整资料档案，涵盖：

- **硬件层**：ESP32 传感器节点、吸顶灯、应急灯
- **软件层**：火灾逃生系统（3D建筑模型 + 蚁群算法路径规划 + 动态模拟）
- **学术层**：论文、专利、软著
- **竞赛层**：计算机设计大赛、挑战杯、互联网+、物联网竞赛等
- **实践层**：科普创新、三下乡、攀登计划、粤港澳大湾区项目
